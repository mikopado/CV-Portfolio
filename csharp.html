SportsBar. A social network for sports fans. You can create your own profile, upload posts and comments and build your own network of friends as well as filtering posts based on favourite topics (e.g. team, sport). Application developed using ASP.NET MVC. You can view the code on GitHub or open the app deployed on AppHarbor.

SkiHill. A simple ski products store web app. The application includes also an example of WCF services and REST API. Application developed using ASP.NET MVC. You can view the code on GitHub or open the app deployed on AppHarbor.
