Java Apps

A&E department. A Java desktop application to resemble a accident and emergency department. The application includes a good example of custom Doubly Linked List to implement either a queue data structure or priority queue. The application also includes a custom O/RM to handle the interaction with the underlying database (SQLite). Swing has been used to implement the GUI.

SOLID. This app is an excellent example of the application of SOLID design principles. The program follows the principles for building a more maintainable, extensible and testable application.

Multithreading. An example of client-server multithreaded application using server socket in Java.
